FAQ

Common questions

What is MCP?
Model Context Protocol (MCP) is an open standard that lets AI tools connect to external data sources. Think of it as a USB port for AI — plug in vidIQ, and your AI assistant can access your YouTube data directly.
Does this use my YouTube API quota?
No. vidIQ MCP uses vidIQ's own data infrastructure, not your YouTube API quota. Your channel data is already synced through vidIQ — the MCP just gives your AI tools a way to query it.
Can the AI modify my YouTube channel?
No. MCP access is strictly read-only. Your AI can analyze your data, find trends, and give recommendations — but it can never upload, edit, or delete anything on your channel.
How many API keys can I create?
You can create multiple API keys — one per AI tool, or one to share across tools. Each key can be revoked individually if needed.
Does this cost extra beyond my vidIQ plan?
No additional cost. MCP access is included with the Max plan at no extra charge. You're accessing the same data your vidIQ dashboard already shows — just through a different interface. No per-query fees, no API add-ons.
Which AI tools are supported?
Any tool that supports the MCP standard: Claude (web, desktop, mobile, Code), ChatGPT, Cursor, Codex, OpenClaw, Windsurf, Warp, Gemini CLI, n8n, and more. If it has MCP support, it works with vidIQ.
Is my data safe?
Yes. Your API key is scoped to your account only. AI providers don't store your vidIQ data — they query it in real-time and discard it after generating a response. You can revoke any key instantly from your settings.
